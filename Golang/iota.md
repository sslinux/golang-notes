#iota
#枚举
#常量 


* iota 是 go语言的常量计数器，只能在常量的表达式中使用；
* iota在const关键字出现时将被重置为0.
* const中每新增一行常量声明将被iota计数一次(iota可理解为const语句块中的行索引)
* 使用iota能简化定义，在定义枚举时很有用；
* 


* const iota = 0 // 无类型整数值

* iota是一个预定义的标识符，代表顺序按行增加的无符号整数，每个const声明单元（被括号括起来）相互独立，分别从0开始。


## 几个常见的iota示例：

### 使用下划线(_)跳过某些值：

```go
const (
	n1 = iota   //0
	n2          // 1
	_
	n4          // 3
)

```


### iota声明中间插队

```go
const (
	n1 = iota   // 0
	n2 = 100    // 100
	n3 = iota   // 2
	n4          // 3
)

const n5 = iota // 0
```

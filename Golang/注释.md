#注释

# golang两种注释形式

## 1、行注释

```go
// 注释一行；
// 函数注释
func main() {
	pass
}
```

## 2、块注释(多行注释)

* 基本语法

```go
/*
	多行注释
	内容
*/
```

* 使用细节
  - 对于行注释和块注释，被注释的文字，不会被Go编译器执行；
  - 块注释里面不允许有块注释嵌套；


## 3、规范的代码风格

### 正确的注释和注释风格
* Go官方对剑使用行注释来注释整个方法和语句；

### 正确的缩进和空白
* 使用一次tab操作，实现缩进，默认整体向右边移动，使用shift+tab整体向左移动；；
* 使用[[gofmt]]来进行代码格式化；
* 运算符两边习惯性各加一个空格； 
* 一行最长不超过80个字符，超过的请使用换行展示，尽量保持格式优雅；

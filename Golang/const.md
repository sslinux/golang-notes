#常量 

# const用于定义常量；

相对于变量，常量是恒定不变的值，多用于定义程序运行期间不会改变的那些值。 常量的声明和变量
声明非常类似，只是把 var 换成了 const ，`常量在定义的时候必须赋值。`

* 在定义了常量后，在整个程序运行期间他们的值就不能再发生变化了；

* 多个常量一起声明：

```go
const (
	pi = 3.1315
	e = 2.7182
)
```

* const同时声明多个常量时，如果省略了值则表示和上面一行的值相同；

```go
const (
	n1 = 100
	n2
	n3
# n1,n2,n3的值都是100；
)
```

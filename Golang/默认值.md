#默认值
#零值

> 在go中，数据类型都有一个默认值。当程序员没有赋值时，就会保留默认值，在Go中，默认值又叫零值；


# 基本数据类型的默认值如下

| 数据类型 | 默认值(零值)  |      |
| -------- | ------------- | ---- |
| 整型      | 0             |      |
| 浮点型 | 0 | |
| string   | ""   空字符串 |      |
| 布尔类型  | false        |      |
|          |               |      |
|          |               |      |

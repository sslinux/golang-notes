### 整型使用细节

* Golang的整型分：有符号和无符号，int 和 uint大大小跟系统相关，32位系统是32位，4个字节；64位系统是64位，8个字节；
* Golang的整型默认声明为int型；
```go
package main
import "fmt"

func main() {
	var n1 = 100
	fmt.Printf("n1 的类型是：%T\n", n1)   // 查看变量类型使用格式化输出: %T;
}
```

* 如何在程序中查看某个变量的字节大小和数据类型；
  - 查看变量类型： 格式化输出%T；
  - 查看变量的字节大小，unsafe.Sizeof(n1)
* Go程序中整型变量在使用时，遵守保小不保大的原则，即：在保证程序正确运行下，尽量使用占用空间小的数据类型。
```go
// 存取 年龄：
var age byte = 90
```
* bit: 计算机中的最小存储单位。 byte：计算机中基本存储单元；1byte=8bit;





## 浮点型

| 类型           | 占用存储空间 | 表数范围               |
| -------------- | ------------ | ---------------------- |
| 单精度 flaot32 | 4 byte       | -3.403E38 ~ 3.403E38   |
| 双精度 float64 | 8 byte       | -1.798E308 ~ 1.798E308 |

* 说明：
  * 关于浮点数在机器中存放形式的简单说明， 浮点数=符号位 + 指数位 + 尾数位， 浮点数都是有符号的；
  * 尾数部分可能都是，造成精度损失；float64的精度比float32的要准确，如果要保存高精度的数，选用float64；

### 浮点型使用细节

* 1、Golang浮点类型有固定的范围和字段长度，不收具体OS(操作系统)的影响；
* 2、Golang的浮点型默认声明为float64类型；
* 3、浮点型常量有两种表示形式：
  * 十进制数形式： 如： 5.12 ； 0.521  (必须有小数点)
  * 科学计数法形式：如： 5.1234e2 = 5.12 * 10的2次方； 5.12E-2 = 5.12/10的2次方；
* 4、通常情况下，应该使用float64，因为它比float32更精确，开发中建议使用float64；




# 字符型
> Golang中my没有专门的字符类型，如果要存储单个字符(字母)，一般使用byte来保存；


> 字符串不是一串固定长度的字符连接起来的字符序列。Go的字符串是由单个字节连接起来的。也就是说对于`传统的字符串是由字符组成的`，而`Go的字符串不同，它是由字节组成的`；

```go
package main

import "fmt"
// golang中字符类型的使用
func main() {
    var c1 byte = 'a'
    var c2 byte = '0' //字符的0
    
   	// 当直接输出byte值时，输出的是对应字符的码值；
    fmt.Println("c1=", c1)
    fmt.Println("c2=", c2)
    
    // var c3 byte = '北' // overflow溢出
    var c3 int = '北'
    fmt.Printf("c3=%c, c3对应码值=%d\n", c3, c3)
}
```

* 说明：
  - 如果我们保存的字符在ASCII表中，比如[0-1,a-z,A-Z...] 可以直接保存到byte型变量中；
  - 如果我们保存的字符对应码值大于255，这时可以考虑使用int类型保存；
  - 若干我们需要按照字符的方式输出，需要使用格式化输出：%c 

### 字符类型使用细节
* 1、字符常量是用单引号(' ')括起来的单个字符。例如：var c1 byte = 'a'     var c2 int = '中'     var c3 byte = '9'
* 2、Go中允许使用转义字符 '\'' 来讲其后的字符转变为特殊字符型常量；例如： var c3 char = '\n'
* 3、Go语言的字符使用UTF-8编码，[查询字符的utf8码值](http://www.mytju.com/classcode/tools/encode_utf8.asp)
	* 英文字母-1个字节； 汉字-3个字节
* 4、在Go中，字符的本质是一个整数，直接输出时，是该字符对应的UTF-8编码的码值；
* 5、可以直接给某个变量赋一个数字，然后按格式化输出使用%c，会输出该数字对应的unicode字符；
* 6、字符类型是可以进行运算的，相当于一个证书，因为它都对应有Unicode码；
```go
// 字符类型 运算，按照码值进行运算；
var n1 = 10 + 'a'    // 10 + 97 = 107
fmt.Println("n1=", n1)
```

## 字符类型本质探讨
* 1、字符型 存储到 计算机中，需要将字符对应的码值(整数)找出来，
	*  存储： 字符 --> 对应码值 --> 二进制 --> 存储
	*  读取： 二进制 --> 码值 --> 字符 --> 读取
* 2、字符和码值的对应关系是通过字符编码表决定的(是规定好的)；
* 3、Go语言的编码都统一成了utf-8,再无乱码困扰；

# 字符串
## [[string]]
> Golang官方将string归属到基本数据类型；

> 字符串就是一串固定长度的字符连接起来的字符序列；
> Go的字符串是由单个字节连接起来的；
> GO语言的字符串的字节使用UTF-8编码表示的Unicode文本；

## string使用注意事项和细节
* 1、Go语言的字符串的字节使用UTF-8编码标识Unicode文本，这样GOlang统一使用UTF-8编码，中文乱码问题不再；
* 2、字符串一旦赋值了，字符串就不能修改了： 在Go中字符串是不可变的；
![[Pasted image 20211125231135.png]]
* 3、字符串的两种表示形式：
	* (1) 双引号，会识别转义字符；
	* (2) 反引号，以字符串的原生形式输出，包括换行和特殊字符，可以实现防止攻击、输出源代码等效果；
* 4、字符串拼接方式，使用加号(+)
```go
// 字符串拼接
var str = "hello " + "world"
str += " haha!"
```

* 5、当一行字符串太长时，需要使用到多行字符串，可以如下处理：
```go
// 当一个拼接的操作很长时，怎么办，可以分行写，但是需要见+号保留在上一行；
str4 := "hello " + "world" + "hello " +
"golang" + 
"Jason"
fmt.Println(str4)
```
